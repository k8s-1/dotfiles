# Manages $HOME configuration files
This project contains a dotfile baselayer.
Usage: `stow --restow .`

## Setup
1. Adjust .gitconfig

2. To generate jira cli config,
```
jira init
```
Add it from
~/.config/.jira/.config.yml

See https://github.com/ankitpokhrel/jira-cli
